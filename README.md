# 陈老师问前程

一个基于传统六爻占卜的在线算卦网站。

## 功能特点

1. **问题选择**：提供6种常见问题的分类选择
2. **铜钱起卦**：模拟传统三枚铜钱抛掷过程
3. **卦象生成**：根据抛掷结果自动生成初始卦和变卦
4. **卦象展示**：显示卦名和卦辞信息

## 使用方法

1. 打开 `index.html` 文件
2. 从下拉框中选择要询问的问题
3. 点击"开始算卦"按钮
4. 依次点击"抛掷铜钱"按钮6次（每次模拟抛掷3枚铜钱）
5. 系统自动生成初始卦和变卦，并显示相关信息

## 起卦规则

- **1背（一背两字）**：单，点一点，表示阳爻（不动）
- **2背（两背一字）**：拆，点两点，表示阴爻（不动）
- **3背**：重，画一圈，表示阳动（初始卦为阳，变卦为阴）
- **3字**：交，打一叉，表示阴动（初始卦为阴，变卦为阳）

## 文件说明

- `index.html` - 主页面结构
- `style.css` - 样式文件
- `script.js` - 核心逻辑实现
- `问题清单.md` - 问题分类列表

## 技术实现

- 纯前端实现，无需服务器
- 使用原生HTML/CSS/JavaScript
- 响应式设计，支持移动端访问
